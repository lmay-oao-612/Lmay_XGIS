# XGIS 程序介绍

## 1. 项目概述
XGIS 是一个基于 .NET Framework (C# WinForms) 开发的轻量级桌面地理信息系统（GIS）原型程序。本项目旨在提供一个基础的 GIS 框架，支持空间数据的可视化、属性编辑及基础的几何运算，适用于地理信息算法验证或小型行业应用开发。

## 2. 核心模块说明
- BasicClasses.cs: 核心几何对象定义（点、线、面）及坐标转换算法。
- Form1.cs: 地图主视图窗口，处理图形渲染（GDI+）、缩放、平移及用户交互。
- FormAttribute.cs: 属性数据管理界面，实现空间要素与业务属性的关联展示。
- Program.cs: 程序启动入口。

## 3. 主要功能
- 矢量要素渲染：利用 GDI+ 实现点、线、面等基础地理几何图形的绘制。
- 地图交互操作：支持地图的平移（Pan）与缩放（Zoom）等基础交互。
- 属性数据管理：提供表格化界面，支持查看和编辑地理要素的非空间业务属性。

## 4. 后续开发建议
- 数据持久化：增加对 Shapefile、GeoJSON、KML 等标准 GIS 数据格式的支持。
- 渲染优化：引入图层管理（TOC）机制，并考虑使用硬件加速（如 SkiaSharp）提升渲染性能。
- 空间分析：增加缓冲区分析（Buffer）、叠加分析（Overlay）等空间拓扑算法。
- 架构完善：引入多线程加载机制，优化空间索引结构以支持大规模数据处理。
